K-Means Clustering Visualisation

This interactive project demonstrates how the K-Means clustering algorithm works under different initialization strategies (random, k-means++, etc.). It visualizes the iterative updates of centroids and cluster assignments in 2D, offering an intuitive understanding of convergence behavior. The goal was to compare how initialization impacts final clusters and learning speed.

Singular Value Decomposition (SVD) on MNIST

I applied Singular Value Decomposition (SVD) to reduce the dimensionality of the MNIST handwritten digits dataset before feeding it into a logistic regression classifier. I evaluated classification accuracy across retained singular values and visualized the compressed digits to show the trade-off between compression and performance.

Latent Semantic Analysis (LSA) Search Engine

I built a document retrieval system using Latent Semantic Analysis. Users input a query, and the system ranks documents based on cosine similarity in reduced semantic space. This involved TF-IDF vectorization, SVD, and bar chart visualizations of relevance scores.

Linear Regression Visualisation
